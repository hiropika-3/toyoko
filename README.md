# 声と言葉で伝える自信を育てるアプリ

「何を話すか」より、「どう話すか」で心が動く、話し方のスキル向上をサポートするアプリケーションです。

## 概要

このアプリケーションは、話し方のスキルを向上させたい方々をサポートするツールです。対話形式で伝えたい内容を整理し、効果的な話し方を提案します。また、音声録音と解析機能により、あなたの声の特徴を分析し、改善点を提案します。

### 対象ユーザー

- 会社や学校でスピーチを頼まれる立場の人や講師
- 司会、ナレーターなどのプロを目指す人
- SNSなどで話すことに挑戦したい人
- 子供の話す力を育てたい保護者
- 面接やプレゼンで"印象に残る話し方"を身に着けたい人

## 機能

1. **対話形式での情報収集**
   - 「誰にどんな気持ちを届けたいか」を対話形式で聞き出します
   - ユーザーの伝えたい想いを引き出す質問を提示します

2. **話し方の提案**
   - 収集した情報から話材（エピソード・ネタ）を提案します
   - 聞き手の心が動く文章例を生成します
   - ユーザーの個性に合ったトーンや話法を提案します
   - 「間の入れどころ」「感情がのるワード」などに印を付けて表示します

3. **音声解析とフィードバック**
   - 提案した文章の読み上げ録音機能を提供します
   - リアルタイム声解析と音声の可視化を行います
   - 声解析結果からのフィードバックとアドバイスを提供します

## インストール方法

1. リポジトリをクローンまたはダウンロードします。

```bash
git clone https://github.com/yourusername/toyoko.git
cd toyoko
```

2. 必要なパッケージをインストールします。

```bash
pip install -r requirements.txt
```

## 使用方法

1. アプリケーションを起動します。

```bash
python run_app.py
```

2. ブラウザが自動的に開き、アプリケーションのインターフェースが表示されます。

3. 声を解析する
   - 「録音」ボタンをクリックして話し始め、「停止」ボタンをクリックして録音を終了します
   - 「停止」ボタンをクリックすると、あなたの声の特徴が分析され、フィードバックが表示されます

## 技術スタック

- Python 3.9+
- Gradio：GUIインターフェース
- NumPy, SciPy：音声処理
- Matplotlib：音声可視化
- librosa：音声特徴抽出
